## Spacing Scale

Declared values (multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding |
| sm | 8px | Compact element spacing |
| md | 16px | Default element spacing |
| lg | 24px | Section padding |
| xl | 32px | Layout gaps |
| 2xl | 48px | Major section breaks |
| 3xl | 64px | Page-level spacing |

**Exceptions from Phase 1 Dashboard.vue:**
- `padding: 20px` — used in dashboard-main, welcome-card margin-bottom (normalize to 20px = acceptable, close to md/lg)
- `gap: 15px` — header-right elements (should standardize to 16px in Phase 2)
- `width: 200px` — sidebar filter (D-16 specifies sidebar, use 200px)

### 1. TopicPool.vue — D-16 Implementation

**Wireframe Description:**
```
┌────────────────────────────────────────────────────────────┐
│ Dashboard Header (Phase 1 inherited)                       │
├──────────────────┬─────────────────────────────────────────┤
│ Sidebar 200px    │ Main Content Area                       │
│ ┌──────────────┐ │ ┌─────────────────────────────────────┐ │
│ │ 领域筛选     │ │ │ el-table                            │ │
│ │ ┌──────────┐ │ │ │ ┌─────────────────────────────────┐ │ │
│ │ │ ○ 全部   │ │ │ │ │ 标题 │ 领域 │ 类型 │ 描述 │ 操作│ │ │
│ │ │ ○ 软件工程│ │ │ │ ├───────────────────────────────┤ │ │
│ │ │ ○ 大数据  │ │ │ │ │ Row 1...                       │ │ │
│ │ └──────────┘ │ │ │ │ Row 2...                       │ │ │
│ │              │ │ │ │ Row N...                       │ │ │
│ │ ┌──────────┐ │ │ └─────────────────────────────────┘ │ │
│ │ │ 提交自拟  │ │ │                                     │ │
│ │ │ 选题按钮  │ │ │                                     │ │
│ │ └──────────┘ │ │                                     │ │
│ └──────────────┘ │ └─────────────────────────────────────┘ │
└──────────────────┴─────────────────────────────────────────┘
```

**Grid Layout:**
- Container: `el-container` with `el-aside` (width: 200px) + `el-main`
- Sidebar: `el-card` with vertical radio-group for domain filter
- Table: `el-table` with full-width, `v-loading` for loading state

**Sidebar Filter Details:**
- Header: "领域筛选"
- Radio group: `el-radio-group` with `el-radio-button`
  - 全部 (value: null) — shows all topics
  - 软件工程 (value: "SE") — filter by domain
  - 大数据 (value: "BD") — filter by domain
- Button: "提交自拟选题" at bottom of sidebar, opens CustomTopicDialog

**Topic Pool Table Columns:**

| Column | Width | Content | Element Plus Component |
|--------|-------|---------|------------------------|
| 标题 | 200px | topic.title | plain text, ellipsis overflow |
| 领域 | 100px | domain display | `el-tag` (primary for SE, success for BD) |
| 类型 | 100px | type display | `el-tag` (default for SYSTEM, warning for CUSTOM) |
| 描述 | flex | topic.description (truncated) | plain text, show full in detail dialog |
| 操作 | 150px | buttons | `el-button` (详情, 选择) |

**Detail Dialog:**
- `el-dialog` width: 600px, title: "选题详情"
- `el-descriptions` :column="1" border
- Fields: 标题, 描述, 背景, 目标, 技术栈建议 (as tags)
- Footer: `el-button` type="primary" "选择此选题"

## Interaction Patterns

### Hover States

| Element | Hover Effect | Duration |
|---------|--------------|----------|
| Project card | shadow="hover" + translateY(-2px) | 0.2s ease |
| Table row | el-table stripe + hover highlight | instant (Element Plus default) |
| Primary button | opacity 0.8 + cursor pointer | 0.15s |
| Danger button | opacity 0.8 + cursor pointer | 0.15s |
| Sidebar radio | checked state highlight | instant |

### Transitions

| Transition | From | To | Duration |
|------------|------|----|----------|
| Dialog open | opacity 0 | opacity 1 | 0.3s |
| Dialog close | opacity 1 | opacity 0 | 0.2s |
| Loading overlay | hidden | visible | instant |
| Tag status change | previous color | new color | 0.2s |

### Loading States

| Component | Loading Indicator | Placement |
|-----------|-------------------|-----------|
| TopicPool table | el-table v-loading | table overlay |
| ProjectList | el-card v-loading | card overlay |
| Form submit | button loading prop | button icon |

## Accessibility Requirements

### Keyboard Navigation

| Component | Key | Action |
|-----------|-----|--------|
| TopicPool table | Tab | Navigate between rows and buttons |
| TopicPool table | Enter | Activate focused button |
| Sidebar filter | Arrow keys | Change radio selection |
| Dialog | Esc | Close dialog |
| Dialog | Tab | Navigate form fields |
| Dialog form | Enter | Submit form (when focus on submit button) |

### ARIA Labels

| Element | aria-label |
|---------|------------|
| Domain filter | "领域筛选" |
| Topic table | "选题列表" |
| 详情 button | "查看选题详情" |
| 选择 button | "选择此选题创建项目" |
| 删除 button | "删除此项目" |
| Custom topic dialog | "提交自拟选题表单" |

### Focus Management

- Dialog open: Focus first required input
- Dialog close: Return focus to trigger button
- Table navigation: Focus follows visual order
- Delete confirmation: Focus cancel button first
