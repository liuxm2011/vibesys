## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-01-01 | Information Disclosure | API key exposure | mitigate | Keep MINIMAX_API_KEY in backend .env only; never expose in frontend or response |
| T-03-01-02 | Tampering | IDOR - generate for other user's project | mitigate | authMiddleware + projectId ownership check in ai.routes.ts |
| T-03-01-03 | Tampering | Prompt injection via topic content | mitigate | System prompt isolation; topic content in user message role; escape special characters if needed |
| T-03-01-04 | Denial of Service | API timeout on long generation | mitigate | Set 30s timeout on AI route; show loading state in frontend |
</threat_model>
